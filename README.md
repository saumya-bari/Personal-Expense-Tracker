# Personal-Expense-Tracker

# Description
This is a simple console-based Personal Expense Tracker application written in Python. It allows users to:

Add new expenses with details such as amount, date, category, and notes.

View all recorded expenses displayed in a neat tabular format with column headers.

Calculate total expenses overall or filtered by category.

Show basic statistics (maximum, minimum, average expense).

Edit and delete existing expense entries by selecting their list number.

The program uses loops for continuous menu navigation, if/else conditions for option handling, and functions (def) for modular code structure.

# Features
User-friendly menu-driven interface.

Input validation for numeric amount fields.

Clear display of expense records with column headers.

Editing and deleting functionality to maintain accurate records.

Easy-to-understand code structure suitable for learning Python basics and real-world application.

# How to Use
Run the Python script.

Choose options from the menu by entering the corresponding number.

Follow on-screen prompts to add, view, edit, or delete expenses.

Enter expense details such as date (YYYY-MM-DD), category, amount, and optional notes.

To exit the program, select option 0.

# Requirements
Python 3.x (compatible with any modern version)
